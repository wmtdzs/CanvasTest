元素创造了一个固定大小的画布，它公开了一个或多个渲染上下文， 其可以用来绘制和处理要展示的内容。我们将会把注意力放在2D渲染上下文中。其他种类的上下文也许提供了不同的渲染方式；比如WebGL 使用了基于OpenGL ES的3D上下文。

canvas起初是空白的。为了展示，首先脚本需要找到渲染上下文，然后在它上面绘制。 元素有一个叫做的方法， 这个方法是用来获得渲染上下文和它的绘画功能。只有一个参数，上下文格式。 

        var canvas = document.getElementById('canvas');
        var ctx = canvas.getContext('2d');

代码第一行通过使用方法来为元素 得到DOM对象。一旦有了元素对象，你可以通过使用它的getContext()方法来访问绘画上下文。

替换内容是用于在不支持标签的浏览器中展示的。 通过简单的测试getContext()方法的存在，脚本可以检查编程支持性。上述代码可变为： 

        var canvas = document.getElementById('canvas');
        
        if (canvas.getContext) {
          var ctx = canvas.getContext('2d');
        } else {
          ...
        }
